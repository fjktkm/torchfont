Getting Started
===============

This section explains how to install and start using TorchFont.

Installation
------------

The package requires Python 3.10+ and PyTorch 2.6+.

Install TorchFont with **uv**:

.. code-block:: bash

   uv add torchfont

Or with **pip**:

.. code-block:: bash

   pip install torchfont

Quick Example
-------------

A minimal example showing how to import and initialize TorchFont:

.. code-block:: python

   import torchfont
